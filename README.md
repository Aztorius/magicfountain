# Magic Fountain
A Fountain editor available on Linux.

[![Build Status](https://travis-ci.org/Aztorius/magicfountain.svg?branch=master)](https://travis-ci.org/Aztorius/magicfountain)

## Install

qmake with Qt5 is required.

### Linux
```
qmake
make
sudo make install
```
